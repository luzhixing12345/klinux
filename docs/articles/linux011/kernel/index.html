从本质上来讲，中断是一种电信号，当设备有某种事件发生时，通过总线把电信号发送给中断控制器

linux当中的中断可以分为两类

  • 硬中断: 由硬件产生的，比如，像磁盘，网卡，键盘，时钟等。每个设备或设备集都有它自己的IRQ（中断请求）。基于IRQ，CPU可以将相应的请求分发到对应的硬件驱动上（注：硬件驱动通常是内核中的一个子程序，而不是一个独立的进程）
